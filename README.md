# 🧮 ACALCU

ACALCU is a modern Windows 11-inspired calculator application built with Python and Tkinter by AKHOURI ANMOL KUMAR.

It combines a sleek frosted UI, customizable buttons, memory functions, history tracking, and lightweight performance into a clean desktop calculator experience.



# ✨ Features

- Modern Windows 11-style UI
- Frosted minimal design aesthetic
- Interactive custom canvas buttons
- Lightweight and fast
- Memory operations
  - MC
  - MR
  - M+
  - M-
  - MS
- Calculation history window
- Edit Mode for button customization
- Customizable buttons:
  - colors
  - shapes
  - text
  - font size
  - dimensions
  - image icons
- Standard arithmetic operations
- Scientific utilities:
  - Square
  - Square root
  - Reciprocal
  - Percentage
  - Sign toggle
- Mouse-based interaction
- Persistent configuration saving
- Image button support using Pillow

# 🎨 Customization System

ACALCU includes a built-in Edit Mode where users can customize buttons in real time.

Supported customization:
- Background color
- Text color
- Font size
- Shapes
- Width/height
- Icons/images
- Custom labels

Configurations are automatically saved locally.

---

# 📜 History System

ACALCU stores:
- Previous calculations
- Scientific operation results
- Expression logs

Includes:
- Scrollable history window
- Clear history support

# 🧠 Scientific Functions

Supported functions:

- Reciprocal (`1/x`)
- Square (`x²`)
- Square Root (`√x`)
- Percentage (`%`)
- Positive/Negative toggle (`+/-`)
